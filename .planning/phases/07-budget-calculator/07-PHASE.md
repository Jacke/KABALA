# Phase 7: Budget Calculator

## Goal
Create a budget calculator that lets users input their current expenses and see what equivalent budget they would need in a target city.

## Dependencies
- Phase 2: Data Layer (city metrics for conversion ratios)

## Plans

### Plan 07-01: Expense input form
Build the expense input interface with categorized fields.

**Deliverables:**
- Source city selector
- Expense input fields by category (rent, food, transport, etc.)
- Pre-fill with average values from selected city
- Clear/reset functionality

### Plan 07-02: Budget conversion and results
Calculate and display equivalent budget in target city.

**Deliverables:**
- Target city selector
- Conversion calculations using city metrics
- Results display with breakdown by category
- Total budget comparison
- Savings/cost difference indicator

## Success Criteria
- Users can input expenses by category
- Pre-fill works from city averages
- Conversion shows equivalent costs in target city
- Clear breakdown of cost differences
- No TypeScript errors
- Build succeeds
